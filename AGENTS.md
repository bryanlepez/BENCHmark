# AGENTS.md --- Master Plan for BENCHmark

## Project Overview

**App:** BENCHmark\
**Goal:** A fast, minimalist macro tracking web app for serious lifters
with no core features behind paywalls.\
**Stack:** Next.js (TypeScript) + Supabase (Auth + Postgres + RLS) +
Vercel\
**MVP Timeline:** 2 weeks\
**Budget:** \<\$50/month

## Plan → Execute → Verify

1.  Plan before coding
2.  Execute one feature at a time
3.  Verify with lint/build/manual checks

## Constraints

-   Web app MVP only
-   Email/password auth (Supabase)
-   Must-have: Food Search, Macro Calculation, Totals, Daily Logs
-   Not in MVP: payments, coaching, native app, social, AI recs
